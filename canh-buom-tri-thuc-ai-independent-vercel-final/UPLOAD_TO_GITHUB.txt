CÁCH TRIỂN KHAI BẢN NÀY

1. Tạo repository GitHub mới hoặc xóa toàn bộ file cũ trong repository hiện tại.
2. Giải nén ZIP.
3. Tải TẤT CẢ file và thư mục bên trong ZIP lên THƯ MỤC GỐC của repository.
   package.json phải nằm ngay ở trang đầu repository, không nằm trong thư mục con.
4. Tạo project Vercel mới từ repository đó.
5. Root Directory: để trống hoặc chọn dấu chấm (.).
6. Framework: Vite.
7. Không cần tự nhập Install Command/Build Command vì vercel.json đã cấu hình.
8. Thêm biến môi trường GEMINI_API_KEY.
9. Deploy.

Log đúng phải có:
- corepack npm ci --no-audit --no-fund
- npm 10.9.2
- corepack npm run build
- Build Completed
