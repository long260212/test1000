1. Tạo repository GitHub MỚI.
2. Tải toàn bộ file của dự án vào THƯ MỤC GỐC repository.
3. Import repository mới vào Vercel.
4. Root Directory: ./
5. Không nhập "npm build" ở Install Command.
6. Nếu bật Override trong Vercel:
   Install Command = npm install
   Build Command   = npm run build
   Output Directory = dist
7. Thêm GEMINI_API_KEY trong Vercel Environment Variables.
8. Redeploy và bỏ dùng build cache nếu Vercel đang giữ cấu hình cũ.
